使用别名模式以任何方式格式化图例键。
    使用别名模式以任何方式格式化图例键。
 

    示例： {{metric.name}}  -  {{metric.label.instance_name}}  

    结果：    cpu / usage_time  -  server1-europe-west-1  
 

      模式

    
    
      
  • 
        {{metric.type}} = 指标类型，例如： compute.googleapis.com/instance/cpu/usage_time
      
    • 
      
  • 
        {{metric.name}} = 将度量的一部分命名为，例如： instance/cpu/usage_time
      
    • 
      
  • 
        {{metric.service}} = 服务部分的度量，例如:  compute
      
    • 
      
  • 
        {{metric.label.label_name}} = 度量标签元数据，例如：metric.label.instance_name
      
    • 
      
  • 
        {{resource.label.label_name}} = 资源标签元数据， 例如：resource.label.zone
